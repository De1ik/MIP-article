\begin{abstract}

ChatGpt Prompts are invaluable tools that helps to increase the efficiency of information search.
Using keywords and intelligently placed hints like:
•	[Imagine you are …]
•	[Attention to …]
User will be able to get a clear answer to almost any question, no matter if it concerns a math equation, legal advice or learning a new programming language. 
Having studied the logic of answer formation, the user will know what the AI needs to emphasize and how to form a correct and optimized queries, which in turn will improve the clarity of the output information and save the time.
\end{abstract}
